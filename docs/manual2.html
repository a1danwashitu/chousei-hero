使い方

  1. 作業するディレクトリを作成する
  2. 調整さんのページから 出欠表をダウンロードする を選択する
  3. ダウンロードの設定はデフォルトのものを選択し、chouseisan.csv を 1. で作成したディレクトリにダウンロードする
  4. ターミナルで
    chousei-hero init {ディレクトリ名}
    を実行する
  5. 作成したディレクトリに出力されたファイルを編集する
    1. members.yaml
      1. 各 member の過去の当番の回数を count に入力する
    2. duties.yaml

      1. 日にちの階層を設定する

        duty1 を day1 の下に配置する

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0
      requiers: 1
  - name: duty1
    child:
    - name: duty1
      requiers: 1

        ->

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0
      requiers: 1
- name: day1
  child:
  - name: duty1
    child:
    - name: duty1
      requiers: 1

      2. 同時に入れない当番を設定する

        duty0-0 と duty0-1 を同時に入れない当番に設定する

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0-0
      requiers: 1
  - name: duty0-1
    child:
    - name: duty0-1
      requiers: 1

        ->

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0-0
      requiers: 1
    - name: duty0-1
      requiers: 1

      3. 当番に必要な人数を設定する

        duty0 の当番に必要な人数を設定する

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0
      requiers: 1

        ->

        - name: day0
  child:
  - name: duty0
    child:
    - name: duty0
      requiers: 3
  6. ターミナルで
    chousei-hero solve {ディレクトリ名}
    を実行する
