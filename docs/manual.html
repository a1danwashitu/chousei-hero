使い方

1. 調整さんのページから `出欠表をダウンロードする` を選択する
2. 「文字コードの設定」は適切なものを、「行と列の設定」は「標準」を選択し、出欠表をダウンロードする。
3. ダウンロードしたファイルを開いて、内容を「csv入力」欄にコピペし、「読み込み」を押す。
4. 「設定入力」に読み込んだ情報が表示されるので、設定を編集する。
4.1. 当番設定
4.1.1 日にちの階層を設定する
    duty1 を day1 の下に配置する
4.1.2 同時に入れない当番を設定する 
    duty0-0 と duty0-1 を同時に入れない当番に設定する
4.1.3. 当番に必要な人数を設定する 
    duty0 の当番に必要な人数を設定する
4.2. メンバー設定
4.2.1 各メンバーの過去に入った当番の回数を count に半角で入力する。
4.3. ステータス設定
4.3.1 必要なら出欠表を編集できます。対応が意味わからんかもですが...
5. 設定の編集が終わったら、「実行」を押す。
6. ページの下に結果が出力される。
